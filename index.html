ABOUT

Hi, and welcome to my website. I put this together myself in order to showcase some of my work in web development, photography, and graphic art. Along with travel, these are my chief interests.

I don't actually get paid to take photos, build websites, or create computer graphics. I do these things for the sheer enjoyment.

One defining characteristic of web development is the dizzying speed at which it evolves. I've been playing around with website building for a long time now and I've seen many things change in that time. On the one hand it's been getting easier to frame up web apps with the multitude of new technologies at our disposal. On the other, it's become more difficult when you consider all of the various screen sizes that have to be taken into consideration in order to ensure that our apps and websites are optimized for viewing on all these devices.

I enjoy learning new things and I do what I can to keep up with new innovations in web development. Periodically I rebuild my portfolio site so that I can make use of some of the latest code writing. For this site I made use of CSS Grid and Flexbox which are used for doing layouts. CSS Grid is new enough that Internet Explorer has yet to support it. Hopefully you're not viewing this on IE.

Since the last time I put together a website I learned some of the new Javascript code called ES6 and I've made use of some it on this site. Javascript is pretty heady stuff. At least it is for me. I haven't fully mastered it yet but I'm gradually getting it down.

I actually have 3 other websites currently on line. There are links to those other sites in the next section. Keep scrolling or use the navigation link. I encourage you to check out my other websites.

If you would like to share feedback about any of my work online just use the contact form. I'd love to hear from you.

Thanks for taking the time to visit.
